# Bonbon Matcha

A modern website for Bonbon Matcha, a premium matcha café in Cupertino, built to showcase the brand, menu, and experience online.

## Live Demo
[Visit the website](https://bonbon-matcha.replit.app)

## What it does
This website gives Bonbon Matcha a polished online presence where visitors can:
- Learn about the brand.
- Explore the café’s matcha-focused identity.
- View menu items or featured offerings.
- Get a feel for the atmosphere before visiting.

## Features
- Clean, mobile-friendly landing page.
- Brand-forward design inspired by matcha culture.
- Menu and highlight sections.
- Simple navigation for visitors.
- Optimized for a smooth visual experience.

## Tech Stack
- Development: Replit, Cursor
- Frontend: React, HTML, CSS
- Styling: Tailwind CSS
- Build tool: Vite
